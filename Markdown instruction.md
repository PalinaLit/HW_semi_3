## Списки

Чтобы добавить ненумеровнные списки, небходимо пукт выделить звездочкой (*). Например, вот так:
* Элемент 1
* Элемент 2
* Элемент 3

Чтобы добавить нумерованные списки, небходимо просто пронемеровать пункты. Например, вот так:
1. Первый пункт
2. Второй пункт
3. Третий пункт
